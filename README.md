## 🏃‍♀️ Run It Locally

1. **Clone this repo:**
   ```sh
   git clone https://github.com/nikmahla/Portfolio-.git
   cd Portfolio-
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

3. **Start the dev server:**
   ```sh
   npm run dev
   ```
   Then open [http://localhost:5173](http://localhost:5173) in your browser.
